# webnote

simple c++ web app

## how to use this project?
make sure you have installed [vcpkg](https://learn.microsoft.com/en-us/vcpkg/get_started/get-started), cmake and ninja.

Configure the build using CMake.
```
cmake --preset=dev
```

Build the project.
```
cmake --build build
```

Run the project.
```
./build/webnote
```
